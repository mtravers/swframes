( content < chunk > )
Gets the results of a bioblog chunk, for example to do continuing computations.

Example:

Chunk #$crx:bioblog/Entry/42/3320: (+ 3 4)
=> 7

To access the result (i.e., the number 7) from that
chunk and use it later:

(* (content #$crx:bioblog/Entry/42/3320) 2)
=> 14
